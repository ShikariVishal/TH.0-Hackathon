#**Language_Translator for documents** _ver 1.0.0_

This project takes the input file and translates it to the required language(currently hardcoded to german language in frontend) which can be viewed on the webpage. 
The main python script named "server.py" is server side file which accepts the requests and returns the response which is rendered on the html page.

Steps to execute the file :
1. To start server run 'python server.py'
2. Go to 'react-app' folder and run 'npm install'
3. To start react app run 'npm run start'

##**Prerequisites**

1. python 3.0 or higher version

Modules :
1. Flask 1.1.2
2. npm 6.14.8
3. translate 3.5.0
4. flask_cors 3.0.9
5. ipfshttpclient 0.6.1
6. requests 2.25.0
